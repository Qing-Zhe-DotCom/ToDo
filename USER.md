# ToDo 用户使用手册

本文档面向当前仓库中的 **源码运行版本**，基线时间为 **2026-04-04**。内容全部以现有代码的真实行为为准。

如果你想先了解项目全貌，请看 [README.md](./README.md)；如果你准备继续维护或开发，请看 [DEV.md](./DEV.md)。

## 1. 当前是什么版本

当前仓库里的 ToDo 不是双击安装即可用的正式发布版，而是一个需要从源码运行的 JavaFX 桌面应用工程。

你现在可以实际使用到的能力包括：

- 日程列表管理
- 时间轴视图
- 热力图视图
- 右侧详情面板
- 设置中心
- 主题切换
- 日程卡片样式切换
- 待办数量驱动的窗口标题 / 任务栏角标更新

当前仍未完成的部分包括：

- 流程图页面
- 登录功能
- 正式安装包 / 发布包

## 2. 启动前提

### 必备软件

- Java 11 或更高版本
- Maven
- MySQL

### 数据库前提

当前程序会连接本地 MySQL：

- 主机：`localhost`
- 端口：`3306`
- 数据库：`todo_db`

而且连接信息目前仍然是硬编码在源码中的。如果你的本地数据库环境与源码不一致，程序会无法连接。

程序启动后会尝试自动保证 `schedules` 表及部分字段存在，但前提仍然是数据库连接本身成功。

## 3. 如何启动当前版本

在仓库根目录打开终端后执行：

```bash
cd todo
mvn clean compile javafx:run
```

如果你只是想确认测试现状，可以执行：

```bash
cd todo
mvn test
```

注意：当前 `mvn test` 还没有恢复到稳定可用状态，完整测试会在启动阶段失败，详见 [README.md](./README.md) 的测试状态章节。

## 4. 主界面布局说明

程序启动后，界面主要分为三部分。

### 左侧边栏

负责：

- 切换主视图
- 搜索日程
- 展开“更多功能”区域

你会看到这些主入口：

- 日程管理
- 日程时间轴
- 日程热力图
- 日程流程图

底部“更多功能”区域展开后可见：

- 新建日程
- 登录
- 设置
- 退出

如果你没看到“设置”或“新建日程”，请先展开底部的“更多功能”。

### 中央内容区

用于显示当前主视图：

- 日程列表
- 时间轴
- 热力图
- 流程图占位页

### 右侧详情面板

当你选中某条日程后，右侧会显示详情面板。它支持查看、编辑、删除和切换完成状态。

关闭方式：

- 点击面板外空白区域
- 按 `Esc`

## 5. 全局搜索怎么用

左侧搜索框按回车后，会切换到日程列表并显示搜索结果。

当前版本的搜索规则是：

- 只搜索名称和描述
- 只有输入非空关键字并按回车时才会触发搜索
- 清空搜索框本身不会自动恢复默认列表

如果你想回到普通列表状态，最稳妥的方式是重新进入“日程管理”视图并重新加载列表。

## 6. 日程管理（列表页）

这是当前最完整的主工作台。

### 6.1 可以做什么

- 查看当前日程
- 新建日程
- 按条件筛选
- 按不同规则排序
- 切换完成状态
- 查看详情

### 6.2 新建日程入口

你可以通过两个入口新建：

- 左侧边栏“更多功能”中的“新建日程”
- 某些视图底部的“新建日程”按钮

### 6.3 列表页控制项

列表顶部目前有这些控制：

- `显示过去日程`
- 排序方式：按日期 / 按优先级 / 按分类
- 筛选方式：全部 / 未完成 / 已完成 / 高优先级 / 即将到期

### 6.4 `显示过去日程` 的真实含义

如果 **不勾选** `显示过去日程`：

- 已过期但未完成的日程会被隐藏
- 截止日期在未来 7 天之后的日程也会被隐藏

所以默认列表更像“近期待办面板”，而不是无限完整列表。

### 6.5 待办 / 已完成分组

列表会把内容拆成两个区域：

- 待办
- 已完成

`已完成` 区域支持折叠。切换完成状态时，列表会播放卡片收缩、重排、落位之类的过渡动画。

### 6.6 完成状态怎么切换

每张卡片左侧都有一个环形状态控件。

你可以：

- 点击它把未完成改为已完成
- 点击已完成项把它改回未完成

当前版本在切换时会有预览、回弹、迁移等动效反馈，所以你会看到比传统复选框更明显的状态变化。

## 7. 新建 / 编辑弹窗怎么用

当前版本使用同一个弹窗处理“新建”和“编辑”。

### 7.1 可填写字段

- 名称
- 描述
- 开始日期
- 截止日期
- 优先级
- 分类
- 标签
- 颜色标记
- 提醒开关
- 提醒日期
- 提醒时间

### 7.2 当前校验规则

- 名称不能为空
- 截止日期不能为空
- 如果填写了开始日期，则开始日期不能晚于截止日期

校验不通过时，对应输入区域会显示错误样式，保存不会成功。

### 7.3 提醒设置

提醒不是独立通知系统，而是作为日程字段保存下来。你可以在弹窗中决定是否启用提醒，并选择日期和时间。

## 8. 右侧详情面板怎么用

当你选中某条日程时，右侧面板会展示：

- 标题
- 当前状态
- 开始 / 截止日期
- 优先级
- 分类
- 标签
- 提醒时间
- 描述

你可以直接在这里执行：

- 编辑
- 删除
- 标记完成 / 标记未完成

状态文案目前可能显示为：

- 已完成
- 已过期
- 即将到期
- 进行中

## 9. 时间轴视图怎么用

点击左侧“日程时间轴”进入。

### 9.1 当前真实实现

当前时间轴不是旧文档里那种“日 / 周 / 月 / 全部”切换视图，而是按日程持续时间自动分组：

- 短期日程：`< 7 天`
- 中期日程：`7 - 35 天`
- 长期日程：`> 35 天`

### 9.2 顶部功能

时间轴顶部提供：

- 开始日期
- 结束日期
- 重置视角

它们用于缩放当前想查看的时间范围。

### 9.3 交互方式

- 鼠标滚轮可控制横向滚动
- 如果当前范围包含今天，视图会尽量自动滚到今天附近
- 点击卡片：打开右侧详情面板
- 双击卡片：打开编辑弹窗
- 点击卡片里的状态控件：切换完成状态

### 9.4 卡片样式

时间轴里的日程卡片会受到“设置 -> 样式”中的全局卡片风格配置影响。

## 10. 热力图视图怎么用

点击左侧“日程热力图”进入。

### 10.1 支持的视图模式

- 周视图
- 月视图
- 年视图

默认打开月视图。

### 10.2 顶部按钮

热力图顶部目前提供：

- 周 / 月 / 年切换按钮
- 上一周期
- 回到今天
- 下一周期

### 10.3 如何理解颜色深浅

颜色越深，表示该日期对应的“已完成任务数”越多。

这里统计的不是“当天覆盖了多少任务”，而是“有多少已完成任务的完成时间落在这一天”。

### 10.4 点击日期后的效果

点击某个热力图单元格后：

- 该日期会被选中
- 下方会显示“该日期的日程”卡片列表

这里展示的是**覆盖该日期**的日程，不要求它们一定在这一天完成。

### 10.5 下方日程卡片

下方面板中的卡片支持：

- 查看基本信息
- 点击打开详情面板
- 双击打开编辑弹窗
- 点击卡片里的状态控件切换完成状态

当你把卡片标记为完成时，热力图当前版本会给“已完成区域”一个明显的接收反馈动画。

## 11. 流程图页面当前是什么状态

点击左侧“日程流程图”后，你会进入一个占位页面。

当前真实状态是：

- 有标题
- 有“开发中”提示
- 没有真正的流程图节点、连线或布局逻辑

不要把它当成已完成功能使用。

## 12. 设置中心怎么用

设置入口位于左侧边栏底部的“更多功能”区域里。

当前设置中心包含 3 个页面：

- 详情
- 主题
- 样式

### 12.1 详情页

用于查看当前应用状态与当前生效的视觉配置，例如：

- 当前主题
- 当前日程卡片样式

### 12.2 主题页

当前可切换 8 套内置主题：

- 浅色
- 薄荷
- 海洋
- 落日
- 薰衣草
- 森林
- 石板
- 马卡龙

此外还支持导入外部 CSS 主题。

主题色卡是**点击即预览并保存**的交互。

### 12.3 样式页

样式页用于切换全局日程卡片风格，影响：

- 日程管理
- 时间轴
- 热力图

当前内置 7 种风格：

- 经典实体卡片
- 清新扁平
- 温馨治愈风
- 现代高级极简风
- 新粗野主义
- Material You
- 拟物浮雕风

样式页的选择会在你点击设置对话框的 `OK` 后保存并刷新各视图。

## 13. 当前版本未支持 / 与直觉不一致的地方

- 流程图仍是占位页
- 登录功能仍是占位提示
- 没有正式安装包或稳定发布版 `jar`
- 数据库连接仍需依赖本地 MySQL，且配置写在源码里
- 搜索不会因为你把输入框清空就自动恢复默认列表
- 热力图的颜色统计和下方列表展示的是两个不同维度

## 14. 常见问题（FAQ）

### 14.1 程序启动后提示数据库连接失败怎么办？

请先检查：

- MySQL 是否已经启动
- `todo_db` 是否存在
- 本地连接信息是否与源码中的硬编码配置一致

### 14.2 为什么我新建了日程，但在列表里没看到？

常见原因有：

- 你没有勾选“显示过去日程”
- 该任务的截止日期离今天太远，超出了列表默认关注范围
- 当前列表仍停留在搜索结果状态

### 14.3 为什么时间轴里看不到某条任务？

时间轴只会稳定显示有有效日期的任务。请确认：

- 是否填写了开始日期或截止日期
- 该任务是否落在当前顶部日期范围内

### 14.4 为什么热力图某天颜色很浅，但下方列出了很多任务？

因为两个维度不同：

- 颜色看的是当天完成了多少任务
- 下方列表看的是哪些任务覆盖了这一天

### 14.5 为什么我没看到设置、新建、退出这些按钮？

这些按钮位于左侧边栏底部的“更多功能”区域。请先展开它。

## 15. 结语

当前版本已经可以完成日程录入、查看、状态切换、时间轴浏览和热力图复盘，但它仍然是一个“源码可运行版本”，不是打磨完成的正式成品。

如果你是普通使用者，建议优先使用：

- 日程管理
- 时间轴
- 热力图
- 设置中心

如果你准备继续完善它，请转到 [DEV.md](./DEV.md)。
