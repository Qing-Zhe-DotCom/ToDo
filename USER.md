## 快速开始

### 安装与启动
本项目是一个基于 JavaFX 开发的桌面客户端程序，零技术背景用户也可以通过以下步骤快速运行：
1. 确保您的电脑上已安装 **Java 11 或更高版本**。
2. 双击运行系统提供的 `.jar` 可执行文件或通过启动脚本启动程序。
3. 启动后，您将直接进入“日程管理”主界面，可立即点击右上角的 **“+ 新建日程”** 开始规划您的任务。

## 核心功能说明

### 1. 日程管理 (基础列表)
这是您查看和管理所有日程的基础界面，提供类似 Excel 表格的清晰列表。
- **操作步骤**：
  1. 点击左侧导航栏的 **“日程管理”**。
  2. 点击右上角 **“+ 新建日程”**，在弹窗中输入日程名称、起止时间、优先级等信息并保存。
  3. 在列表中，您可以直接点击复选框标记任务为“已完成”，或点击“编辑/删除”按钮管理现有日程。
- **使用场景**：适合快速录入任务、进行批量状态更新或查阅任务的详细文字描述。

### 2. 日程时间轴 (Timeline)
时间轴是本系统的特色模块，它能将您的日程化作长短不一的“卡片”，平铺在横向的时间坐标系上。
- **功能特色**：
  - **全周期智能分层**：系统会自动将您的日程分为 **短期 (<7天)**、**中期 (7-35天)** 和 **长期 (>35天)** 三个独立区域，上下滑动鼠标滚轮即可全局俯瞰。
  - **平滑滚动**：在时间轴区域内上下滚动鼠标滚轮，可平滑地左右拖拽时间轴。
  - **多风格切换**：通过顶部的下拉框，您可以将卡片切换为“经典实体卡片”、“现代毛玻璃”、“极简线框”、“扁平马卡龙”或“赛博霓虹”等不同视觉风格。
- **操作步骤**：
  1. 点击左侧导航栏的 **“日程时间轴”**。
  2. 使用顶部的“开始日期”和“结束日期”选择器来缩小或放大您想查看的时间段。
  3. 鼠标悬停在卡片上，卡片会自动放大并浮起；点击卡片可在右侧面板查看详情。

### 3. 日程热力图 (Heatmap)
热力图类似于 GitHub 的代码提交记录图，用颜色的深浅直观展示您每天完成任务的数量。
- **功能特色**：颜色越深，代表您在该日期完成的任务越多。
- **使用场景**：适合用于年终总结或月度复盘，直观感受自己的工作密度和产出效率。

### 4. 日程流程图 (Flowchart)
流程图模块将所有待办事项按**优先级**和**时间顺序**自动串联成节点网络。
- **功能特色**：高优先级任务会被优先排在前面，已完成的任务节点会变为绿色，超期未完成的节点会标红。
- **使用场景**：当您面对一堆繁杂的任务不知从何下手时，打开流程图，它会告诉您第一步该做什么。

## 辅助功能与个性化

- **全局主题切换**：点击顶部菜单栏的 **“选择CSS风格”**，可在“亮色模式 (Light)”和“暗色模式 (Dark)”间自由切换，保护视力。
- **全局搜索**：在左上角的搜索框中输入关键字，可快速过滤左侧边栏的内容。
- **数据本地存储**：所有日程数据均保存在您本地的 SQLite 数据库（`todo.db`）中，确保您的隐私绝对安全。

## 常见问题与故障排查

| 现象 | 可能的原因 | 解决指引 |
| --- | --- | --- |
| **程序点击无反应/闪退** | Java 运行环境未正确安装。 | 请前往 Oracle 或 Adoptium 官网下载安装 Java 11+ JRE。 |
| **时间轴中找不到刚刚新建的日程** | 1. 日程没有设置开始/截止日期。<br>2. 日程日期超出了顶部工具栏选择的“日期范围”。 | 1. 为日程补充完整的日期。<br>2. 点击顶部工具栏的“重置视角”按钮恢复默认全局视图。 |
| **中文显示为乱码或方块** | 系统缺失常用中文字体。 | 确保您的操作系统安装了微软雅黑 (Microsoft YaHei) 或类似标准字体。 |

- **重构**：全新升级的时间轴 (TimelineView) 模块，实现全时段智能垂直分层（短/中/长期日程分离）。

- **修复**：修复了在特定操作下可能导致的数据刷新不同步问题。
