# ToDo 用户使用手册

本文档描述的是当前仓库中源码运行版本的真实状态；未来已确定的改造路线会单列说明，但不会把未来目标写成当前已实现能力。

如果你想先了解项目全貌，请看 [README.md](./README.md)。  
如果你准备继续维护，请看 [DEV.md](./DEV.md)。  
如果你想看更细的视图说明，请看 [UI_VIEWS.md](./UI_VIEWS.md)。

## 1. 当前这是一个什么版本

当前仓库里的 ToDo 不是正式安装包，也不是已发布桌面成品，而是一个需要从源码运行的 JavaFX 桌面应用。

你当前可以实际使用的能力包括：

- 日程列表管理
- 时间轴视图
- 热力图视图
- 右侧详情面板
- 设置中心
- 主题切换
- 日程卡片样式切换
- 待办数量驱动的窗口标题 / 任务栏角标更新

当前仍未完成或仍是占位的部分：

- 日程流程图页面
- 登录功能
- 正式安装包 / 发布包

## 2. 启动前提

### 必备软件

- Java 21
- Maven
- 本地 MySQL

### 当前数据库事实

当前默认运行仍依赖本地 MySQL 配置。  
默认配置值来自 [`todo/src/main/resources/application-defaults.properties`](./todo/src/main/resources/application-defaults.properties)：

- 主机：`localhost`
- 端口：`3306`
- 数据库：`todo_db`
- 用户：`root`
- 密码：空字符串

如果你的本地数据库环境与默认配置不一致，程序可能在加载列表、时间轴或热力图时弹出数据库错误。

## 3. 如何启动当前版本

推荐在仓库根目录打开 PowerShell 后执行：

```powershell
cd .\todo

$env:JAVA_HOME='C:\Program Files\Microsoft\jdk-21.0.10.7-hotspot'
$env:JAVA21_HOME=$env:JAVA_HOME
$env:PATH="$env:JAVA_HOME\bin;$env:PATH"

mvn -f .\pom.xml clean javafx:run
```

如果你只想先确认是否能编译：

```powershell
cd .\todo
mvn -f .\pom.xml clean compile
```

如果你只想跑测试：

```powershell
cd .\todo
mvn -f .\pom.xml test
```

## 4. 主界面布局

程序启动后，界面主要分为三部分：

- 左侧边栏
- 中央主内容区
- 右侧详情面板

### 左侧边栏

左侧边栏负责：

- 切换主视图
- 搜索日程
- 展开“更多功能”区域

当前可见主入口：

- 日程管理
- 日程时间轴
- 日程热力图
- 日程流程图

底部“更多功能”展开后可见：

- 新建日程
- 登录
- 设置
- 退出

### 中央主内容区

中央区域会显示当前主视图：

- 日程列表
- 时间轴
- 热力图
- 流程图占位页

### 右侧详情面板

选中某条日程后，右侧会打开详情面板。  
它支持查看详细信息、编辑、删除和切换完成状态。

关闭方式：

- 点击详情面板外空白区域
- 按 `Esc`

## 5. 搜索怎么用

左侧搜索框输入关键字后按回车，会切换到日程列表并显示搜索结果。

当前搜索规则：

- 只在名称和描述中搜索
- 只有输入非空关键字并按回车时才会触发
- 清空搜索框本身不会自动恢复默认列表

## 6. 日程管理（列表页）

这是当前最完整的主工作台。

### 当前可以做什么

- 查看当前日程
- 新建日程
- 通过筛选缩小结果范围
- 切换排序方式
- 切换完成状态
- 打开详情面板

### 当前可见筛选与排序要点

当前代码里至少存在这些筛选概念：

- 全部
- 未完成
- 已完成
- 高优先级
- 即将到期

列表也保留了“显示过去日程”之类的近期待办控制逻辑，因此默认列表并不是无限完整清单，而更像一个近期工作台。

### 完成状态

每张卡片左侧都有状态控件，你可以直接切换完成 / 未完成。  
当前版本会带有视觉反馈和列表重排动画。

## 7. 新建 / 编辑弹窗

当前版本使用同一个弹窗处理“新建”和“编辑”。

### 当前可填写字段

- 名称
- 描述
- 开始日期
- 截止日期
- 优先级
- 分类
- 标签
- 颜色标记
- 提醒开关
- 提醒日期
- 提醒时间

### 当前校验要点

- 名称不能为空
- 截止日期不能为空
- 如果填写了开始日期，则开始日期不能晚于截止日期

## 8. 右侧详情面板

详情面板当前会展示：

- 标题
- 当前状态
- 开始 / 截止日期
- 优先级
- 分类与标签
- 提醒时间
- 描述

你可以直接在这里执行：

- 编辑
- 删除
- 完成 / 标记未完成

## 9. 时间轴视图

点击左侧“日程时间轴”进入。

### 当前真实能力

- 顶部有开始日期、结束日期和“重置视角”
- 鼠标滚轮可以横向滚动
- 视图会尽量滚到今天附近
- 点击卡片可打开详情
- 双击卡片可打开编辑弹窗
- 点击卡片状态控件可切换完成状态

### 当前真实分组方式

当前时间轴不是旧设计里那种“日 / 周 / 月 / 全部切换组件”，而是按日程持续时间自动分组：

- 短期日程：`< 7 天`
- 中期日程：`7 - 35 天`
- 长期日程：`> 35 天`

## 10. 热力图视图

点击左侧“日程热力图”进入。

### 当前支持的模式

- 周视图
- 月视图
- 年视图

### 顶部功能

- 周 / 月 / 年切换
- 上一周期
- 回到今天
- 下一周期

### 如何理解颜色

颜色越深，表示该日期对应的已完成任务数越多。  
下方面板则显示“覆盖该日期”的任务列表，这两个维度并不完全相同。

## 11. 流程图页面当前状态

“日程流程图”目前仍是占位页。  
代码里明确写着“日程流程图功能正在开发中...”，不要把它当成已完成功能使用。

## 12. 设置中心

设置入口在左侧边栏底部“更多功能”区域里。

当前设置中心包含 3 个页面：

- 详情
- 主题
- 样式

### 详情页

用于查看当前应用与视觉配置，例如：

- 当前主题
- 当前日程卡片样式

### 主题页

当前支持内置主题切换，并支持导入外部 CSS 主题文件。

### 样式页

用于切换全局日程卡片样式，影响：

- 日程管理
- 时间轴
- 热力图

## 13. 登录当前状态

当前“登录”按钮还不是正式账号功能。  
点击后会弹出信息提示：“登录功能开发中...”。

## 14. 常见问题

### 14.1 程序启动后提示数据库连接失败怎么办？

先检查：

- MySQL 是否已经启动
- `todo_db` 是否存在
- 当前配置是否仍使用默认值
- 本地账号密码是否与配置匹配

### 14.2 为什么我运行 Maven 时提示找不到 `pom.xml`？

优先使用：

```powershell
mvn -f .\pom.xml clean javafx:run
```

不要完全依赖某些特殊 PowerShell 路径表示下的当前目录。

### 14.3 为什么流程图打不开真正内容？

因为它目前仍是占位页。

### 14.4 为什么登录没有进入账号系统？

因为登录功能当前还只是占位提示。

### 14.5 为什么我新建了任务却没看到？

常见原因有：

- 当前列表仍处于搜索结果状态
- 当前列表筛选隐藏了它
- 该任务不在当前时间轴或热力图的可视范围内

## 15. 当前事实与后续路线

### 当前事实

- 当前源码运行版默认仍依赖本地 MySQL
- 当前可用主视图是列表、时间轴、热力图和详情面板
- 流程图和登录尚未完成

### 已确定的后续路线

- 本地数据库会转向 SQLite
- 云端同步会以 PostgreSQL + API/Sync 服务为基线
- 日程模型会逐步升级到支持分钟级时间、全局 ID 和同步字段
