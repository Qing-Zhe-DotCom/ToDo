# ToDo 用户使用手册

本文档面向当前仓库中的**源码版本**，基线时间为 **2026-04-03**。内容全部以现有代码实际行为为准。

如果你希望先了解项目全貌，请先看 [README.md](./README.md)；如果你准备继续开发，请看 [DEV.md](./DEV.md)。

## 1. 这是什么版本

当前仓库中的 ToDo 不是“安装后直接双击就能使用”的正式发布版，而是一个可以从源码运行的桌面应用工程。

你现在能实际使用到的核心功能包括：

- 日程列表管理
- 时间轴视图
- 热力图视图
- 右侧详情面板
- 设置中心
- 主题切换
- 时间轴卡片样式切换

目前仍未完成的内容包括：

- 流程图页面
- 登录功能
- 正式安装包 / 发布版

## 2. 启动前提

在当前源码版本中，启动前请先确认以下条件：

### 必需软件

- Java 11 或更高版本
- Maven
- MySQL

### 数据库前提

当前程序会连接本地 MySQL：

- 主机：`localhost`
- 端口：`3306`
- 数据库：`todo_db`

并且源码中使用的是**硬编码连接信息**。如果你的本地数据库环境和源码不一致，程序将无法连接。

程序启动后会尝试自动创建 `schedules` 表，并补齐部分字段；但前提仍然是数据库连接本身已经成功。

## 3. 如何启动当前版本

在仓库根目录打开终端后执行：

```bash
cd todo
mvn clean compile javafx:run
```

如果你只是想验证当前版本是否能通过自动化测试，可以执行：

```bash
cd todo
mvn test
```

当前仓库没有正式提供：

- 可直接双击运行的安装包
- 稳定发布版 `.jar`
- 一键启动脚本

## 4. 主界面布局说明

程序启动后，界面主要分为三部分：

### 左侧侧边栏

负责：

- 切换视图
- 搜索日程
- 展开更多功能

你会看到以下入口：

- 日程管理
- 日程时间轴
- 日程热力图
- 日程流程图

此外还有两个特殊按钮：

- 顶部折叠按钮：可收起 / 展开侧边栏
- 底部“更多功能”按钮：展开后会显示“新建日程、登录、设置、退出”

注意：

- “更多功能”区域默认不是一直展开的
- 如果看不到“设置”或“新建日程”等底部按钮，请先点击侧边栏底部的展开按钮

### 中央内容区

用于显示当前视图：

- 日程列表
- 时间轴
- 热力图
- 流程图占位页

### 右侧详情面板

当你点击某个日程时，右侧会弹出详情面板，用来查看或操作该日程。

它支持：

- 查看标题、日期、优先级、分类、标签、提醒、描述
- 编辑
- 删除
- 标记完成 / 取消完成

关闭方式：

- 点击面板外空白区域
- 按 `Esc`

## 5. 全局搜索怎么用

左侧搜索框可以搜索日程，但当前版本有几个要点：

- 只搜索**名称**和**描述**
- 搜索后会直接切换到“日程管理”列表视图
- 当前版本没有特别完善的“清空搜索后恢复默认列表”交互

如果你搜索后结果和预期不一致，建议：

- 重新输入新的关键词再回车
- 或重启应用后重新进入列表

## 6. 日程管理（列表页）

这是当前版本最完整的主工作台。

## 6.1 进入方式

点击左侧的 **“日程管理”**。

## 6.2 你能做什么

- 查看当前日程
- 新建日程
- 按条件筛选
- 按不同规则排序
- 切换完成状态
- 查看详情

## 6.3 新建日程

你可以通过两个入口新建：

- 列表页右下角 / 右侧的“新建日程”按钮
- 侧边栏“更多功能”面板中的“新建日程”

点击后会打开统一的编辑弹窗。

## 6.4 列表页筛选与排序

列表页顶部有几个控制项：

### `显示过去日程`

勾选后，会显示更多历史任务。

不勾选时，列表更偏向“近期工作面板”，会隐藏：

- 已过期但未完成的旧任务
- 截止日期在未来 7 天之后的任务

### 排序方式

当前支持：

- 按日期排序
- 按优先级排序
- 按分类排序

### 筛选方式

当前支持：

- 全部
- 未完成
- 已完成
- 高优先级
- 即将到期

## 6.5 待办 / 已完成分组

列表会把项目自动分成两组：

- 待办
- 已完成

每组前面都有可折叠的组标题。点击箭头可以收起或展开整组内容。

## 6.6 快速切换完成状态

每条日程左侧都有状态图标。

点击它可以直接：

- 把未完成改为已完成
- 把已完成改回未完成

切换后：

- 列表会刷新
- 右侧详情面板状态会刷新
- 窗口标题 / 任务栏待办数量也会更新

## 6.7 查看详情

点击任意一条日程：

- 列表会选中该项
- 右侧详情面板会弹出

## 7. 新建 / 编辑弹窗怎么用

当前版本的新建与编辑使用同一个弹窗，只是编辑时会自动回填已有数据。

## 7.1 可填写字段

| 字段 | 说明 |
| --- | --- |
| 名称 | 必填，不能为空 |
| 描述 | 可选，多行文本 |
| 开始日期 | 可选 |
| 截止日期 | 必填 |
| 优先级 | 高 / 中 / 低 |
| 分类 | 可选，默认会回落到“默认” |
| 标签 | 可选，普通文本 |
| 颜色标记 | 从预设颜色中选择 |
| 提醒开关 | 可开启或关闭 |
| 提醒日期 | 开启提醒后可选 |
| 提醒时间 | 开启提醒后可选 |

## 7.2 当前校验规则

- 名称不能为空
- 截止日期不能为空
- 如果同时填了开始日期和截止日期，则开始日期不能晚于截止日期

如果校验不通过：

- 输入框会出现错误样式
- 保存按钮不会正常提交

## 7.3 编辑已有日程

当前版本可通过两种方式进入编辑：

- 在右侧详情面板点击“编辑”
- 在时间轴或热力图中**双击**日程卡片

## 8. 右侧详情面板怎么用

只要当前选中了日程，右侧详情面板就会显示它的完整信息。

## 8.1 显示内容

- 标题
- 当前状态
- 开始日期 / 截止日期
- 优先级
- 分类
- 标签
- 提醒时间
- 描述

## 8.2 可执行操作

- 编辑
- 删除
- 标记完成 / 标记未完成

## 8.3 状态含义

当前可能看到的状态有：

- 已完成
- 已过期
- 即将到期
- 进行中

## 9. 时间轴视图怎么用

点击左侧 **“日程时间轴”** 进入。

## 9.1 这个页面现在真正做了什么

当前时间轴会把有日期的日程按持续时长分成三个垂直区域：

- 短期日程：少于 7 天
- 中期日程：7 到 35 天
- 长期日程：超过 35 天

这是一种**自动分组展示**，不是“日 / 周 / 月”可切换模式。

## 9.2 页面顶部功能

页面顶部有：

- 开始日期
- 结束日期
- 重置视角

用途：

- 用来缩小或放大当前想看的时间范围
- 如果开始日期晚于结束日期，视图会提示范围无效

## 9.3 浏览方式

时间轴支持：

- 横向拖动滚动条
- 在时间轴区域滚动鼠标滚轮进行横向浏览

如果当前范围包含今天，页面会尽量自动滚动到今天附近。

## 9.4 卡片交互

鼠标移动到卡片上时：

- 卡片会轻微放大
- 卡片会上浮
- 层级会提高，避免被其他卡片遮住

点击卡片时：

- 打开右侧详情面板

双击卡片时：

- 打开编辑弹窗

## 9.5 时间轴样式切换入口

当前版本的时间轴卡片样式切换**不在时间轴顶部**，而是在设置中心里。

可选样式有 7 种：

- 经典实体卡片
- 清新扁平
- 温馨治愈风
- 现代高级极简风
- 新粗野主义
- Material You
- 拟物浮雕风

## 10. 热力图视图怎么用

点击左侧 **“日程热力图”** 进入。

## 10.1 三种查看模式

热力图当前支持：

- 周视图
- 月视图
- 年视图

默认打开月视图。

## 10.2 顶部按钮

页面顶部提供：

- 周 / 月 / 年切换按钮
- 上一周期
- 回到今天
- 下一周期

周期的含义如下：

- 周视图：按周跳转
- 月视图：按月跳转
- 年视图：按年跳转

## 10.3 如何理解颜色深浅

颜色越深，表示该日期对应的“已完成任务数”越多。

这里的统计规则不是“这一天有多少任务”，而是：

- 只统计 `completed = true` 的任务
- 按任务 `updated_at` 的日期来算完成日

所以你在热力图上看到的颜色，代表“完成记录”，不是简单的日程堆积量。

## 10.4 点击日期后的效果

点击某个热力图单元格后：

- 该日期会被选中
- 下方会出现“该日期的日程”卡片列表

这个列表展示的是**覆盖该日期**的日程，不要求这些日程一定是在这一天完成的。

## 10.5 下方日程卡片

下方卡片会显示：

- 名称
- 日期信息
- 优先级
- 分类
- 描述摘要
- 当前状态

点击卡片：

- 打开右侧详情面板

双击卡片：

- 打开编辑弹窗

## 11. 流程图页面当前是什么状态

点击左侧 **“日程流程图”** 后，你会进入一个占位页。

当前真实状态是：

- 只有标题和“开发中”提示

所以不要把它当成已完成功能使用。

## 12.1 当前设置中心包含的页面

主要用于快速确认当前视觉配置。

- 薄荷

- 落日
- 薰衣草
- 森林

修改后会在确认保存后刷新各视图。

## 13. 当前版本未支持 / 与直觉不符的地方

为避免误会，这里单独列出当前版本的限制：

- 流程图仍处于开发中，只是占位页
- 登录功能只是提示“开发中”
- 没有正式安装包或稳定发布版 `jar`
- 没有真正深色主题模式
- 时间轴样式切换入口在设置中心，不在时间轴顶部
- 时间轴是按任务时长自动分组，不是“日 / 周 / 月”切换版
- 搜索只查名称和描述，且当前搜索恢复默认列表的交互不够完善

## 14. 常见问题（FAQ）

## 14.1 程序启动后提示数据库连接失败怎么办？

先检查：

- MySQL 是否启动
- `todo_db` 是否存在
- 你的本地连接信息是否和源码中的硬编码配置一致

当前源码版本不会弹出友好的数据库配置向导，连接不一致时只能修改本地环境或调整源码。

## 14.2 为什么我新建了日程，但在列表里没看到？

常见原因有：

- 你没有勾选“显示过去日程”
- 该任务的截止日期离今天太远，超出了列表默认关注范围
- 当前列表正在显示搜索结果，而不是默认列表

建议先：

- 检查筛选与排序
- 重新进入列表页
- 确认是否仍停留在搜索结果状态

## 14.3 为什么时间轴里看不到某条任务？

时间轴只会稳定显示有有效日期的任务。

请检查：

- 是否填写了开始日期或截止日期
- 该任务是否落在当前顶部日期范围内

如果当前范围被缩得太窄，可以点击“重置视角”。

## 14.4 为什么热力图某天颜色很浅，但下面列出了很多任务？

因为两个维度不同：

- 颜色看的是这一天完成了多少任务
- 下方列表看的是有哪些任务覆盖了这一天

所以“任务很多”不一定等于“完成很多”。

## 14.5 为什么我没看到设置、新建、退出这些按钮？

这些按钮位于左侧侧边栏底部的“更多功能”区域。

如果当前侧边栏是折叠状态或底部功能区未展开，你需要先点击对应展开按钮。

## 14.6 主题导入失败怎么办？

当前外部主题导入只接受 CSS 文件。

如果导入失败，常见原因有：

- 文件不是 `.css`
- CSS 内容不兼容当前 JavaFX 样式类
- 文件路径不可访问

建议先用内置主题确认程序样式系统正常，再逐步调试外部 CSS。

## 14.7 中文显示异常或乱码怎么办？

建议先确认：

- 系统已安装常见中文字体，例如微软雅黑
- 终端或编辑器采用 UTF-8 编码

当前仓库文档已经统一按 UTF-8 重写，但你的本地显示环境仍可能影响观感。

## 15. 结语

当前版本已经可以完成日常的任务录入、查看、时间轴浏览和热力图复盘，但它仍然是一份“源码可运行版本”，不是完全打磨好的成品。

如果你是普通使用者，建议把重点放在：

- 列表页
- 时间轴
- 热力图
- 设置中心

如果你准备继续完善它，请转到 [DEV.md](./DEV.md) 查看架构和工程现状。
