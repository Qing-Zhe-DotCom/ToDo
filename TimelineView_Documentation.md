# 日程时间轴组件 (TimelineView) 说明文档

## 组件简介
`TimelineView` 是一个基于时间轴的日程卡片可视化组件。它以动态按天/周/月为单位的水平时间轴作为基础，支持日程卡片根据其持续天数在对应日期区间内错落堆叠展示。

## 核心功能
1. **三级时间轴与自动分配系统**：
   - **日级时间轴**：以日为单位，自动筛选并显示时长小于 7 天的日程。
   - **周级时间轴**：以周为单位，自动筛选并显示时长在 7-35 天内的日程。
   - **月级时间轴**：以月为单位，自动筛选并显示时长大于 35 天的长周期日程。
   - **全部视图**：混合显示所有日程。
2. **多主题视觉风格**：
   - **经典实体卡片**：使用明显的阴影和实体填充，展现出立体感。
   - **现代毛玻璃**：采用半透明填充、圆角和强色强调，更具现代感。
   - **极简线框**：去除背景填充，仅使用线条边框来标识卡片边界。
3. **交互与动画**：
   - **平滑悬停与置顶**：鼠标悬停在卡片上时，卡片会伴随平滑过渡动画（缩放及向侧边滑出）被提升至最顶层显示。
   - **水平平滑滚动**：支持通过鼠标滚轮进行带有边界检测的水平平滑滑动。
4. **智能排版与重叠**：同一天内如果有多个日程重叠，组件会自动分配层级，使其错落堆叠排列，呈现出类似实体文件夹标签页的视觉效果。

## 演示示例场景

要验证四种典型日程场景（单天、跨天、跨周、跨月），可以在系统中新建以下日程：
- **单天日程**：开始时间 `2026-04-01`，截止时间 `2026-04-01`。（在日级视图中显示）
- **跨天日程**：开始时间 `2026-04-01`，截止时间 `2026-04-03`。（在日级视图中显示）
- **跨周日程**：开始时间 `2026-04-01`，截止时间 `2026-04-10`。（在周级视图中显示）
- **跨月日程**：开始时间 `2026-03-25`，截止时间 `2026-05-15`。（在月级视图中显示）

## API接口与核心类说明

### `TimelineView` (实现 `View` 接口)
主视图类，负责界面初始化、事件绑定以及绘制逻辑。

#### 核心渲染方法：
- `private void drawTimeline() throws SQLException`
  负责执行时间轴视图的完整重新渲染。该方法将获取所选时间范围内的日程，计算各日程对应的显示坐标(startX)和宽度(width)，并将它们绘制到 `timelinePane` 容器中。

#### 核心静态辅助方法：
- `static List<TimelineEntry> buildTimelineEntries(List<Schedule> schedules, LocalDate minDate, LocalDate maxDate, String level)`
  构建渲染实体，执行过滤和自动错落层级分配。
- `static List<Schedule> filterAndSortSchedules(List<Schedule> schedules, String level)`
  根据当前选择的时间轴层级（日、周、月）自动筛选对应时长的日程。

## 响应式与性能
- 组件使用 `ScrollPane` 实现了平滑的水平鼠标滚轮事件监听。
- 对于超过可视范围的部分日程元素只计算和渲染相交区域，大幅减少了不必要的节点绘制，提高了大数据量下的渲染性能。
