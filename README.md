# ToDo 日程管理系统

欢迎使用 ToDo 日程管理系统！这是一个基于 JavaFX 与 SQLite 构建的轻量级、跨平台的桌面日程管理客户端。

本项目旨在提供极简、高效、直观的日程任务管理体验，帮助您从繁杂的待办事项中解脱出来。我们拥有特色的**三级智能分层时间轴 (Timeline)**、直观的工作效率**热力图 (Heatmap)** 以及任务**流程图 (Flowchart)**。

## 🚀 快速启动

1. 确保您的电脑已安装 **Java 11 或以上版本**以及 **Maven** 工具。
2. 克隆本仓库到本地。
3. 在项目根目录（包含 `pom.xml` 的目录）打开终端，运行：
   ```bash
   mvn clean compile javafx:run
   ```
4. 开始享受您的日程规划之旅！
